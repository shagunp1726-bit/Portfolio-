📁 ASSETS FOLDER — Image Upload Guide
======================================

Place your images in this folder with the exact filenames listed below:

─── PROFILE PHOTO ───────────────────────────────────────
  File: profile.jpg  (or profile.png / profile.webp)
  Used in: About section (avatar circle)
  Size: 400×400 px recommended, square crop

─── CERTIFICATION IMAGES ────────────────────────────────
  File: cert-cloud.jpg        → Managing Cloud Infrastructure & Security Analyst
  File: cert-network.jpg      → Network Security Analyst
  File: cert-ibm.jpg          → Web Development Fundamentals (IBM)
  File: cert-mobile.jpg       → Mobile Application Penetration Testing Analyst
  File: cert-cyber.jpg        → Cyber Security Analyst
  File: cert-malware.jpg      → Malware Analysis Analyst
  Used in: Certifications section (hover overlay on each card)
  Size: Any — they will be shown in a popup/overlay

NOTE: If an image is missing, the card will gracefully fall back
      to the placeholder style automatically. No errors will show.
